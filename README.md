Tech Stack
- Supabase
- React + TypeScript
- Vite
- (Preferably css modules)
- React Router

### 1. Clone the Repository

```bash
git clone https://github.com/TCKxIncorporated/ICS321-Tournament
```

### 2. Install Dependencies
```bash
cd Tournament-App
npm install
```

### 3. Start the App
```bash
npm run dev
```
The app will run at http://localhost:5173
